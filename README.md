## Convertible bond

Telegram打新债信息通知机器人

### 环境初始化

```bash
$ python -V
Python 3.7.0

# 安装依赖
$ pip install -r requirements_dev.txt
```

### 配置环境变量

$ echo 'BOT_TOKEN=xxxxxxxxxxxxxxx' >> .env
$ echo 'CHANNEL_ID=xxxxxxxxxxxxxxx' >> .env
